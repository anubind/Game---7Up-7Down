This Python program implements an engaging "7 Up/Down" betting-style game where players predict outcomes relative to the number 7. Players choose from three options via simple menu selection (a/b/c), and the system generates a random number between 2 and 12 to determine wins or losses. The game tracks total matches played and wins, providing real-time statistics and a final summary upon exit. Designed for console execution, it uses a while True loop for continuous play until the user opts out.
